# archive-siteapp-leyu

本仓库用于归档和发布若干独立 HTML 页面。这些页面以静态文件形式存放，不依赖后端服务，可直接通过浏览器打开或部署到任意静态托管平台。

## 项目简介

`archive-siteapp-leyu` 是一个纯前端静态页面集合。页面内容主要涉及信息展示、工具类应用或文档归档，不针对特定域名或网站。仓库本身不包含数据库、服务器端脚本或第三方 API 调用，所有资源均为本地化 HTML/CSS/JS 文件。

## 目录结构

```
archive-siteapp-leyu/
├── index.html          # 入口页面（如需要）
├── pages/              # 存放各独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
├── docs/               # 文档或说明文件（可选）
└── README.md           # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，互不依赖。
- `assets/`：存放页面共用的静态资源，避免重复引用。
- `docs/`：可存放使用说明、变更日志等附加文档。

## 页面归档说明

- 本仓库仅用于归档历史页面或存放临时性演示页面，不保证持续更新或功能完善。
- 页面内容可能包含示例数据、占位文案或测试功能，请勿直接用于生产环境。
- 如需引用或二次开发，请自行评估代码安全性及合规性。
- 仓库不存储任何用户数据，不收集隐私信息。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来改进页面样式、修复错误或增加新页面。
- 提交新页面时，请将其放入 `pages/` 目录，并在入口文件（如 `index.html`）中添加导航链接（如有）。
- 公共资源请优先放入 `assets/` 对应子目录，避免页面间重复打包。
- 所有页面应保持独立运行，避免依赖仓库之外的资源（如外部 CDN 链接），以确保离线可用性。
